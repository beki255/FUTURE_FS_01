# Bereket Sahlemariam - Portfolio Website

A modern, responsive portfolio website built with React.js and Vite.

## Features

- **Modern Design** - Clean, professional UI with smooth animations
- **Responsive** - Works perfectly on desktop and mobile devices
- **Dark/Light Mode** - Toggle between themes
- **Animated Hero** - Letter-by-letter text animation
- **Contact Form** - EmailJS integration for sending messages
- **Downloadable CV** - Easy access to resume

## Sections

- Home
- About
- Skills (MEARN Stack)
- Portfolio

## Getting Started

### Prerequisites
- Node.js installed

### Installation

```bash
# Clone the repository
git clone https://github.com/beki255/portfolio.git

# Navigate to project
cd my_portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

Build the project and deploy the `dist` folder to:
- **Netlify** - Drag and drop dist folder
- **Vercel** - `npx vercel`
- **GitHub Pages** - Configure in settings
